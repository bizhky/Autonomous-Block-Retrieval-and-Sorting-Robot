# Autonomous-Block-Retrieval-and-Sorting-Robot
Arduino Code for 4 Wheels and a Servo Motor connected to a PlayStation 2 controller. Uses the &lt;Servo.h> and &lt;PS2X_lib.h> libraries. Created by a Sophomore at Drury University.

• Collaborated with a team of four engineering students to design, manufacture, and program a robotic system capable of collecting and sorting multiple block geometries.
• Utilized SolidWorks to design custom structural components which were manufactured using 3D printing and laser-cut acrylic fabrication methods.
• Designed the robot chassis and developed precision mounting layouts for mechanical and electrical integration.
• Programmed Arduino-based motor control systems and integrated PS2 controller functionality.
• Contributed to the development of a rack-and-pinion lifting mechanism and claw assembly used for object manipulation.
• Successfully achieved design requirements including five-foot linear travel in both directions and precise 360-degree zero-point turning.
• Helped deliver a fully functional robotic system capable of efficiently locating, collecting, and depositing objects into a custom-built container.
